** 概要
   
   放送大学のインターネット環境を体験的に概観します。

   膨大な情報と過剰なサービスが存在するウェブ、さまざまな情報機器から
   利用できるクラウドとは何か、体験的に学びます。

   多くの情報の中から、自分の情報を確立させる技術が情報リテラシーです。
   情報リテラシーを高めるための考え方と技術を、体験的に学びます。

   講義の前半で、メモと思考のためのツールを紹介します。講義内容をメモ
   し、考えながら受講してください。
